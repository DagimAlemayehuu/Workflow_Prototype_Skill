# workflow-expert-skills Agent

**Expert AI Agent specializing in building flawless automation and AI workflows using n8n-mcp and langflow-mcp servers.**

---

## 🎯 Capability Overview

This agent is powered by a high-fidelity "Expert Brain" containing 14 complementary skills for both n8n and Langflow. It solves common workflow engineering problems:

- **Platform Selection**: Knowing when to use n8n (logic) vs Langflow (reasoning).
- **Tool Precision**: Using native MCP tools efficiently instead of raw API calls.
- **Flawless Validation**: Interpreting runtime and build errors with high accuracy.
- **Architectural Standards**: Building production-ready, scalable patterns.

---

## 📚 Active Skill Set

### Logic & Integration (n8n)
1. **n8n Expression Syntax**: Core syntax and `$json.body` gotchas.
2. **n8n MCP Tools Expert**: Precision tool selection for searching and editing.
3. **n8n Workflow Patterns**: Proven production architectures.
4. **n8n Validation Expert**: Interpreting runtime and build errors.
5. **n8n Node Configuration**: Operation-aware parameter mapping.
6. **n8n Code JavaScript**: High-performance data processing.
7. **n8n Code Python**: Standard library usage and limitations.

### Reasoning & AI (Langflow)
1. **Langflow Core**: JSON structure and handle naming conventions.
2. **Langflow MCP Expert**: Programmatic orchestration via native tools.
3. **Langflow Patterns**: RAG, Multi-Agent, and Tool-use architectures.
4. **Langflow Validation**: Debugging build jobs and execution failures.
5. **Component Configuration**: Deep-dive into LLM and Vector Store settings.
6. **Variables & State**: Global secrets and session-based memory management.
7. **Multi-Platform Bridge**: Orchestrating hybrid (n8n + Langflow) systems.

---

## 🚀 How to Use

Simply interact with your AI agent (Claude Code, Gemini CLI, Jules, etc.) and ask questions related to n8n or Langflow. The relevant skills will activate automatically to guide the agent's behavior.

**Example Queries:**
- "Build a webhook to Slack workflow in n8n."
- "How do I configure a RAG flow in Langflow?"
- "Why is my n8n expression failing?"
- "Create a hybrid system using both n8n and Langflow."
